+LIFETIME OS (oggyfoxy, 21) — v2
+Start: 2025-12-15 (Europe/Paris)
+
+0) CORE AXIOMS (NO NEGOTIATION)
+- I do not trade long-term agency for short-term relief.
+- I do not “check” feelings with compulsions (porn, pulse, scrolling).
+- I act on gates + principles, not on fear.
+- Consistency beats intensity. Simple beats clever.
+- Environment > willpower. Sleep > hacks.
+
+1) HARD BANS (CATEGORY 1: NEVER)
+Substances:
+- no alcohol
+- no recreational drugs
+- no nicotine (tobacco/vape/pouches)
+- no gambling
+
+Sex / validation:
+- no porn (any format)
+- no masturbation
+- no casual sex (hookups/fwb/situationships)
+- no paid sexual content/services (onlyfans/escorts/strip clubs/etc.)
+- no sexual/romantic DMs for validation outside a committed relationship
+- no “testing” the addiction (testing == relapse)
+
+Reassurance-seeking:
+- no symptom googling
+- no pulse/heart checking rituals
+- no looping “am I okay?” scans
+
+2) DAILY INVARIANTS (CATEGORY 2A: MUST, MINIMUM SET)
+- Wake time fixed: ±30 min, 7/7.
+- Phone stays out of bedroom. Bed is sleep + paper book only.
+- Deep work before dopamine:
+  - 90 minutes focused work before social/email/news/feeds.
+  - Phone in another room during the block.
+
+3) WEEKLY QUOTAS (CATEGORY 2B: TARGETS, FLEXIBLE)
+Training:
+- Lift heavy 3–4x/week (tracked progressive overload).
+- Zone 2 cardio 3x/week (~45 min).
+
+Cold / discomfort:
+- Cold exposure 5–7x/week (30s+ cold finish OR equivalent).
+
+Nutrition:
+- Protein target daily (1.6–2.2 g/kg).
+- Mostly whole foods by default.
+- IF 16:8 is a tool, not a religion: aim 4–7x/week if it helps.
+
+4) INTERRUPT PROTOCOLS (CATEGORY 3: WHEN IT HITS)
+4.1 Porn/validation urge (<= 3 minutes, no thinking)
+1) STOP. Label: “urge”.
+2) Write one line: time + emotion + what I’m avoiding.
+3) Surf 90 seconds: observe, don’t negotiate.
+4) Replace (pick ONE):
+   - 5–10 min walk outside (default)
+   - 20 pushups / 2 min wall-sit
+   - cold shower (only if it calms, not if it spikes panic)
+5) Return to the next tiny task (2 minutes). Start the timer.
+
+4.2 Health anxiety spiral (the rule is: no data-seeking)
+1) STOP. Label: “anxiety, not data”.
+2) NO metrics. NO pulse. NO googling.
+3) Do one downshift:
+   - 5–10 min walk, or
+   - light mobility + slow nasal breathing
+4) Re-enter life via action: one concrete task for 2 minutes.
+
+4.3 Focus collapse
+1) Brain dump to paper (max 2 minutes).
+2) Classify:
+   - fatigue -> 10 min break, then restart 25 min
+   - avoidance -> do 5 more minutes, then reassess
+   - overstimulated -> phone away + water + short walk, then restart
+
+5) MEDICAL SAFETY GATES (CARE IS ALLOWED; REASSURANCE IS NOT)
+Principle:
+- I don’t seek care to reduce anxiety. I seek care when gates are met.
+
+Gates (examples; if unsure, err on safety):
+- chest pain/pressure with exertion, fainting, severe shortness of breath
+- new neuro deficits (one-sided weakness, facial droop, speech trouble)
+- black/tarry stools with weakness/dizziness, vomiting blood
+- severe allergic reaction (swelling, wheeze)
+- high fever or worsening symptoms that persist and meaningfully impair function
+
+Rule:
+- If a gate is met -> contact one trusted clinician / emergency services.
+- Otherwise -> run the anxiety protocol and continue the day.
+
+6) ENVIRONMENT DEFAULTS (MAKE RELAPSE INCONVENIENT)
+- Phone: remove feeds/reels/explore. If an app is kept, it’s for messaging only.
+- Bedroom: no charger inside. Blackout + cool room.
+- Workspace: one-tab rules for deep work. Blockers allowed.
+- Social: maintain real contact weekly (loneliness is a relapse trigger).
+- Accountability: one person knows Category 1 rules.
+- Travel: rules persist; vacation is not a moral holiday.
+
+7) WEEKLY REVIEW (SUNDAY, 20–30 MIN)
+Track only what changes behavior:
+- Deep work sessions completed
+- Category 1 clean days (yes/no)
+- Training sessions (lift + zone2)
+- Sleep consistency (wake time compliance)
+- # urges surfed (win/loss)
+- # anxiety spirals (and what triggered them)
+Change ONE variable max for next week.
+Metrics (HRV, etc.) live here only, not during spirals.
+
+8) RESET RULES (NO DRAMA)
+- Category 1 violation:
+  - write: trigger -> sequence -> patch to environment
+  - reset the relevant counter
+  - resume immediately (no binge, no shame loop)
+- Category 2A miss:
+  - log it
+  - protect next day’s wake time + first deep work block
+- Category 2B miss:
+  - adjust schedule; don’t moralize it
+
+9) NORTH STAR (WHY THIS EXISTS)
+- clarity > novelty
+- agency > compulsion
+- output > noise
+- long life, strong body, calm mind
+

postscript:

  • category 1 is the line. category 2a is the minimum viable day. category 2b is weekly throughput.
  • if a day fails, only patch the environment. don’t negotiate the bans.
  • metrics belong in review; anxiety gets protocols, not data.
